# Orbit Telegram Bot

A futuristic assistant for social media downloads and crypto radar.

## Features
- Media: Detect links from TikTok, YouTube, etc. Download MP4/MP3.
- Crypto: Real-time quotes, top gainers, updates, alarms.

## Setup
1. Get token from @BotFather.
2. Set BOT_TOKEN in Replit secrets.
3. Install: `pip install -r requirements.txt`
4. Run: `python main.py`

## Commands
/scan, /grab, /pulse, /winner, etc.

Data from CoinGecko.
